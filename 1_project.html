Static Website with Nginx and Docker

Having just graduated, I wasn’t quite sure where to start showcasing my work. While taking my Data Science SQL course, I followed @amigoscode’s tutorials in parallel and found him to be an excellent instructor. I had bookmarked his and TechWorld with Nana’s “Docker and Kubernetes – Full Course for Beginners” months earlier and started casually watching it to build a stronger foundation in Docker. That’s what ultimately inspired the idea of creating my own website.

Running Containers with Docker

I built and deployed a personal static website using Nginx as the web server and Docker for containerization. I customized the site’s HTML and CSS files to create a professional portfolio layout using Nvim. 


# Create and run a new container
docker run -it -d --name my_container ubuntu bash

# Enter the container interactively
docker exec -it my_container bash

# List files inside
ls

I then configured an Nginx server inside a lightweight Alpine Linux image to serve the pages. The project was version-controlled and hosted through GitHub, demonstrating how containerized web applications can be developed, tested, and shared efficiently.
